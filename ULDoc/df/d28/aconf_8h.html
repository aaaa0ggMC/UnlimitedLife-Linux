配置文件语法：
以分号作为分割符，数据空格间隔开直接平铺写下去，可以使用""进行包装，例子：

header "你好 世界！" "fu\n" { child0 "天"; } { child1 "地" } 有点怪;

解析结果为: header – [array]:{"你好 世界！","fu\n","有点怪"} – child0 -— [array]:{"天"} – child1 -—[array]:{"地"}

可以嵌套{},然后就没了 另外，你无法给root节点添加array item.

item 1 1 1 1;
item2 1 1 1 1;

这两个最终是 [root.(这个没必要写，这里是形象化)]item 和 [root.]item2 而不是root的array中的项目 重复写item可以合并内容
